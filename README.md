# Simter Projects

## 📂 Список проектов
- [Multimodel](projects/multimodel) — система распределения запросов между GPT-5 и GPT-5-nano

## 🚀 Как использовать
Склонировать репозиторий и перейти в нужный проект:

```bash
git clone https://github.com/AAA11A/Simter_projects.git
cd Simter_projects/projects/multimodel
```

Установить зависимости и запустить:

```bash
pip install -r requirements.txt
python src/main.py
```
