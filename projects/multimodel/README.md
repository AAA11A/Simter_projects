# Проект «Мульти модель»

## 🎯 Что это за программа?
**Мульти модель** — это интерактивный интерфейс (работает в Google Colab), который позволяет задавать вопросы искусственному интеллекту и получать ответы от разных языковых моделей (LLM).

Программа автоматически анализирует вопрос:
- Если вопрос **лёгкий** → используется модель **GPT-5-nano**.
- Если вопрос **сложный** → используется модель **GPT-5**.

Выбранная модель возвращает ответ пользователю.

---

## 📊 Что показывает программа?
После каждого запроса отображается:
- **Ответ модели** — текстовый результат.
- **Метрики и стоимость** — таблица с параметрами:
  - этап (classifier / answer),
  - модель,
  - входные токены,
  - выходные токены,
  - время выполнения,
  - стоимость в $,
  - стоимость в ₽,
  - суммарные значения.
- **Классификация** вопроса (лёгкий или сложный).
- **Курс USD→RUB** (через API).
- **Итоговая стоимость** запроса.
- **Дата и время** выполнения.

## ⚙️ Инициализация (первый запуск)
Перед первым использованием необходимо выполнить в отдельной ячейке Colab:

```python
import sys, importlib
sys.modules.pop("src.ui_app", None)

import src.ui_app as ui_app
importlib.reload(ui_app)

ui_app.launch_ui()
```

После этого откроется домашний экран программы.

---

## 🚀 Как пользоваться
1. **Запуск программы**  
   - На домашнем экране доступны кнопки: *«Запустить программу»* и *«Архив запросов»*.  

2. **Вход по паролю**  
   - Введите пароль → нажмите *«Разблокировать»*.  
   - При ошибке появится сообщение.  

3. **Задать вопрос**  
   - Введите текст → нажмите *«Отправить»*.  
   - Программа сама определит сложность и выберет модель.  

4. **Просмотр ответа**  
   - Появится блок с ответом, таблица метрик и информация.  
   - Поле ввода очистится для нового вопроса.  

5. **Циклический режим**  
   - После каждого ответа снова появляется поле для ввода.  
   - Можно продолжать до выхода.  

6. **Архив запросов**  
   - Хранит все вопросы и ответы текущей сессии.  

7. **Завершение работы**  
   - Кнопка *«Выйти из программы»* завершает работу и возвращает на домашний экран.
